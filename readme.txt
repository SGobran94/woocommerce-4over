Woocommerce 4over creates Woocommerce product using 4over api. 


== Description ==

Woocommerce 4over creates Woocommerce products using 4over api. It has option to create either normal woocommerce product or create 4over product.

It is also integrated with multiple shipping address and file uploads before plugin.

== Installation ==

**Installation Instruction & Configuration**    

1. Download the zip file and extract the contents. Upload the 'woocommerce-4over' folder to your plugins directory (e.g, wp-content/plugins/)

2. Before installing this plugin. Install and enable Woocommerce, WooCommerce Ship to Multiple Addresses, Woocommerce uploads, Woocommerce Uploads Before Add-On

4. Copy woocommerce folder from plugin folder to your active theme, make sure you take back up woocommerce folder if already exist in your active theme

4. Activate the plugin through the 'Plugins' menu in WordPress.   

5. Set 4over api credentials under '4over Product Settings' menu from left side

6. Now, go to Woocommerce -> Uploads and do setting as required

7. Now, go to Woocommerce -> Settings -> Shipping - Add shipping method as required

8. Now, go to Woocommerce -> Settings -> Checkout - Enable payment method as required

9. Now, add woocommerce product and select type product type as 4over and then select category and product and Save. First time, it will create product attributes

10. Now, again select same product and update. Now, this time it creates attributes options and assigned to product

11. Now, create bulk variations using woocommerce in-built functionality and add price to any combination so that it can be viewable on front end.

12. Now, view created product in front end and select quantity to display no. of sets

13. Now, select shipping and upload files and do add to cart, then cart and do checkout

14. Now, review order in back end, under Edit order, it will show new tab 'Order Processing and Management'
